Vue指令与原生JavaScript完整映射对照表

🎯 核心概念

Vue概念 原生JavaScript等价操作 说明
数据驱动 手动DOM操作 Vue自动同步数据和视图，JS需要手动更新
响应式系统 事件监听 + 手动更新 Vue使用Proxy/Observer，JS需要getter/setter
组件 函数/类 + DOM操作 Vue组件化，JS需要自己封装复用逻辑

📋 指令映射表

1. 属性绑定 (v-bind)

Vue指令 原生JavaScript 示例对比
v-bind:src="url" element.src = url 图片源绑定
:class="className" element.className = className CSS类绑定
:style="styleObj" Object.assign(element.style, styleObj) 样式对象绑定
:disabled="isDisabled" element.disabled = isDisabled 禁用状态绑定
:href="link" element.href = link 链接绑定
:value="inputValue" element.value = inputValue 表单值绑定 
// Vue写法
<img :src="imageUrl" :alt="altText" />

// JavaScript等价写法
const img = document.createElement('img');
img.src = imageUrl;
img.alt = altText;

2. 事件监听 (v-on)

Vue指令 原生JavaScript 说明
@click="handler" element.addEventListener('click', handler) 点击事件
@submit.prevent="onSubmit" form.addEventListener('submit', e => { e.preventDefault(); onSubmit(); }) 阻止默认提交
@keyup.enter="onEnter" element.addEventListener('keyup', e => { if(e.key==='Enter') onEnter(); }) 回车键监听
@click.stop="onClick" element.addEventListener('click', e => { e.stopPropagation(); onClick(); }) 阻止冒泡
@click.once="onOnce" element.addEventListener('click', onOnce, {once: true}) 只触发一次

3. 条件渲染 (v-if/v-show)

Vue指令 原生JavaScript 区别说明
v-if="condition" if(condition) parent.appendChild(element); else element.remove(); 完全移除/添加DOM
v-show="condition" element.style.display = condition ? 'block' : 'none'; 只控制显示/隐藏
v-else-if="other" else if(other) { /* 显示逻辑 */ } 多条件分支
v-else else { /* 默认显示 */ } 默认分支 
// Vue写法
<div v-if="isLoggedIn">欢迎回来</div>
<div v-else>请登录</div>

// JavaScript等价写法
const container = document.getElementById('container');
if (isLoggedIn) {
    const welcomeDiv = document.createElement('div');
    welcomeDiv.textContent = '欢迎回来';
    container.appendChild(welcomeDiv);
} else {
    const loginDiv = document.createElement('div');
    loginDiv.textContent = '请登录';
    container.appendChild(loginDiv);
}

4. 列表渲染 (v-for)

Vue写法 原生JavaScript等价
v-for="item in items" items.forEach(item => { /* 创建DOM */ })
v-for="(item, index) in items" items.forEach((item, index) => { /* 创建DOM */ })
:key="item.id" element.setAttribute('data-key', item.id) 
// Vue写法
<ul>
  <li v-for="user in users" :key="user.id">
    {{ user.name }}
  </li>
</ul>

// JavaScript等价写法
const ul = document.createElement('ul');
users.forEach(user => {
    const li = document.createElement('li');
    li.textContent = user.name;
    li.setAttribute('data-key', user.id);
    ul.appendChild(li);
});
document.body.appendChild(ul);

5. 双向绑定 (v-model)

表单类型 Vue写法 JavaScript等价
文本输入 <input v-model="text"> input.value = text; input.oninput = e => text = e.target.value
复选框 <input type="checkbox" v-model="checked"> checkbox.checked = checked; checkbox.onchange = e => checked = e.target.checked
单选框 <input type="radio" v-model="picked" value="a"> radio.checked = picked === 'a'; radio.onchange = e => picked = e.target.value
下拉选择 <select v-model="selected"> select.value = selected; select.onchange = e => selected = e.target.value

⚡ 高级特性对照

计算属性 (computed)

// Vue写法
computed: {
  fullName() {
    return this.firstName + ' ' + this.lastName;
  }
}

// JavaScript实现
let _firstName = '';
let _lastName = '';
let _fullName = '';

Object.defineProperty(obj, 'firstName', {
  get() { return _firstName; },
  set(value) {
    _firstName = value;
    updateFullName();
  }
});

function updateFullName() {
  _fullName = _firstName + ' ' + _lastName;
  // 更新相关DOM
}

侦听器 (watch)

// Vue写法
watch: {
  message(newVal, oldVal) {
    console.log(`消息从 ${oldVal} 变为 ${newVal}`);
  }
}

// JavaScript实现
let _message = '';
Object.defineProperty(obj, 'message', {
  get() { return _message; },
  set(newVal) {
    const oldVal = _message;
    _message = newVal;
    console.log(`消息从 ${oldVal} 变为 ${newVal}`);
  }
});
