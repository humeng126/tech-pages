⚡ Vue @ 事件监听语法详解

🎯 @ - 事件监听语法（v-on简写）

💡 作用

@ 是 v-on: 的简写，用于监听DOM事件并执行JavaScript代码

Vue @ 语法
<template>
  <div>
    <!-- 点击事件 -->
    <button @click="handleClick">点击我</button>
    
    <!-- 输入事件 -->
    <input @input="onInput" v-model="text" />
    
    <!-- 键盘事件 -->
    <input @keyup="onKeyUp" />
    
    <!-- 表单提交 -->
    <form @submit="onSubmit">
      <button type="submit">提交</button>
    </form>
    
    <!-- 鼠标事件 -->
    <div @mouseenter="onMouseEnter"
         @mouseleave="onMouseLeave">
      悬停区域
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      text: '',
      clickCount: 0
    }
  },
  methods: {
    handleClick() {
      this.clickCount++;
      console.log('按钮被点击了');
    },
    onInput(event) {
      console.log('输入内容:', event.target.value);
    },
    onKeyUp(event) {
      console.log('按键:', event.key);
    },
    onSubmit(event) {
      event.preventDefault();
      console.log('表单提交');
    }
  }
}
</script>
原生JavaScript等价
// 获取DOM元素
const button = document.querySelector('button');
const input1 = document.querySelector('input[type="text"]');
const input2 = document.querySelector('input[type="search"]');
const form = document.querySelector('form');
const hoverDiv = document.querySelector('.hover-area');

// 数据
let clickCount = 0;
let text = '';

// 添加事件监听器
button.addEventListener('click', function() {
  clickCount++;
  console.log('按钮被点击了');
});

input1.addEventListener('input', function(event) {
  text = event.target.value;
  console.log('输入内容:', event.target.value);
});

input2.addEventListener('keyup', function(event) {
  console.log('按键:', event.key);
});

form.addEventListener('submit', function(event) {
  event.preventDefault();
  console.log('表单提交');
});

hoverDiv.addEventListener('mouseenter', function() {
  console.log('鼠标进入');
});

hoverDiv.addEventListener('mouseleave', function() {
  console.log('鼠标离开');
});

// 手动更新UI（Vue会自动处理）
function updateUI() {
  document.querySelector('.count').textContent = clickCount;
}
等价关系
Vue写法          JavaScript等价

@click="handler"   → addEventListener('click', handler)

@input="onInput"   → addEventListener('input', onInput)

@keyup="onKey"     → addEventListener('keyup', onKey)

@submit="onSubmit" → addEventListener('submit', onSubmit)

@mouseenter="fn"   → addEventListener('mouseenter', fn)

@mouseleave="fn"   → addEventListener('mouseleave', fn)

@focus="onFocus"   → addEventListener('focus', onFocus)

@blur="onBlur"     → addEventListener('blur', onBlur)

@change="onChange" → addEventListener('change', onChange)

@load="onLoad"     → addEventListener('load', onLoad)
🛠️ 事件修饰符详解

💡 什么是修饰符

修饰符是Vue特有的语法糖，用点号 . 连接，简化常见的事件处理需求

Vue修饰符 完整写法 JavaScript等价 说明
@click.prevent v-on:click.prevent event.preventDefault() 阻止默认行为
@click.stop v-on:click.stop event.stopPropagation() 阻止事件冒泡
@click.capture v-on:click.capture addEventListener('click', handler, true) 捕获阶段触发
@click.self v-on:click.self if (event.target === event.currentTarget) 只在元素自身触发
@click.once v-on:click.once addEventListener('click', handler, {once: true}) 只触发一次
@scroll.passive v-on:scroll.passive addEventListener('scroll', handler, {passive: true}) 被动监听，提升性能
Vue修饰符示例
<template>
  <div>
    <!-- 阻止表单默认提交 -->
    <form @submit.prevent="onSubmit">
      <button type="submit">提交</button>
    </form>
    
    <!-- 阻止事件冒泡 -->
    <div @click="parentClick">
      父元素
      <button @click.stop="childClick">
        子元素（不会冒泡）
      </button>
    </div>
    
    <!-- 只触发一次 -->
    <button @click.once="onceClick">
      只能点击一次
    </button>
    
    <!-- 多个修饰符组合 -->
    <button @click.stop.prevent="complexHandler">
      阻止冒泡和默认行为
    </button>
  </div>
</template>
JavaScript手动实现
const form = document.querySelector('form');
const childButton = document.querySelector('.child');
const onceButton = document.querySelector('.once');
const complexButton = document.querySelector('.complex');

// .prevent 修饰符等价
form.addEventListener('submit', function(event) {
  event.preventDefault(); // 手动阻止默认行为
  onSubmit();
});

// .stop 修饰符等价
childButton.addEventListener('click', function(event) {
  event.stopPropagation(); // 手动阻止冒泡
  childClick();
});

// .once 修饰符等价
onceButton.addEventListener('click', function handler() {
  onceClick();
  // 手动移除事件监听器实现只触发一次
  onceButton.removeEventListener('click', handler);
});
// 或者使用 {once: true} 选项
onceButton.addEventListener('click', onceClick, {once: true});

// 多个修饰符组合等价
complexButton.addEventListener('click', function(event) {
  event.stopPropagation();  // .stop
  event.preventDefault();   // .prevent
  complexHandler();
});
⌨️ 按键修饰符

常用按键

@keyup.enter → Enter键
@keyup.tab → Tab键
@keyup.delete → Delete/Backspace
@keyup.esc → Escape键
@keyup.space → 空格键
@keyup.up → 上箭头
@keyup.down → 下箭头
@keyup.left → 左箭头
@keyup.right → 右箭头

系统修饰键

@click.ctrl → Ctrl键
@click.alt → Alt键
@click.shift → Shift键
@click.meta → Meta键(Cmd/Win)
@click.ctrl.exact → 只有Ctrl
@keyup.ctrl.c → Ctrl+C
@keyup.ctrl.s → Ctrl+S

鼠标修饰符

@click.left → 左键点击
@click.right → 右键点击
@click.middle → 中键点击
@contextmenu → 右键菜单
@dblclick → 双击
Vue按键修饰符
<template>
  <div>
    <!-- 回车键搜索 -->
    <input @keyup.enter="search" 
           placeholder="按回车搜索" />
    
    <!-- Ctrl+S保存 -->
    <textarea @keyup.ctrl.s="save"></textarea>
    
    <!-- ESC键取消 -->
    <div @keyup.esc="cancel" tabindex="0">
      按ESC取消
    </div>
    
    <!-- 右键菜单 -->
    <div @contextmenu.prevent="showMenu">
      右键显示菜单
    </div>
    
    <!-- 只有Ctrl键（不能有其他修饰键） -->
    <button @click.ctrl.exact="ctrlOnly">
      只能Ctrl+点击
    </button>
  </div>
</template>
JavaScript按键检测
const input = document.querySelector('input');
const textarea = document.querySelector('textarea');
const div = document.querySelector('.cancelable');
const contextDiv = document.querySelector('.context');
const button = document.querySelector('button');

// 回车键检测
input.addEventListener('keyup', function(event) {
  if (event.key === 'Enter') {
    search();
  }
});

// Ctrl+S组合键检测
textarea.addEventListener('keyup', function(event) {
  if (event.ctrlKey && event.key === 's') {
    event.preventDefault();
    save();
  }
});

// ESC键检测
div.addEventListener('keyup', function(event) {
  if (event.key === 'Escape') {
    cancel();
  }
});

// 右键菜单
contextDiv.addEventListener('contextmenu', function(event) {
  event.preventDefault();
  showMenu();
});

// 精确修饰键检测
button.addEventListener('click', function(event) {
  // 检查只有Ctrl键被按下
  if (event.ctrlKey && !event.altKey && 
      !event.shiftKey && !event.metaKey) {
    ctrlOnly();
  }
});

📊 完整语法对比总结

✅ Vue的优势

  • 语法简洁：@click="handler"
  • 修饰符强大：.prevent, .stop
  • 按键简化：.enter, .ctrl
  • 自动绑定this上下文
  • 声明式，易于理解

⚙️ JavaScript特点

  • 需要手动addEventListener
  • 手动处理event对象
  • 手动检测按键和修饰键
  • 需要管理事件监听器生命周期
  • 更多代码，但更灵活

🎯 使用建议

  • 简单交互 → 用Vue @语法
  • 复杂事件逻辑 → 混合使用
  • 第三方库集成 → 用JavaScript
  • 性能敏感 → 考虑原生JS
  • 学习阶段 → 理解两者对应关系
